Узнал про аргументы командной строки и то как они работают, связывая главную функцию main с командной строкой операционной системы и что через командную строку можно работать с самой программой. 
Пример работы с аргументами командной строки: 
```
int main(int argc, char*argv[]) {
  for(int i = 0; i < argc; ++i) { // используем argc, который отвечает за счетчик аргументов, которые передаем функции main из командной строки, обратившись через путь к main. 
    std::cout << argv[i] << " "; // argv массив строковых аргументов, который включает уже включенные в себя аргументы, написанные в командной строке и выводим эти данные в самой программе
  }
  std::cout << std::endl;
  return  0;
}
```
Далее рассмотрел два типа функций main. Один стандартный, который возвращает int и другой который ничего не возвращает. В сегодняшних проектах использует интовый main, void main раньше был, но не является стандартом.
Можно писать и void main, только программа будет кидать предупреждение о том, что main не будет возвращать исходный результат выполнения программы, что очень важно для самой программы. Лучше использовать int main сегодня.
void main используется только скорее всего для обучения и в реальных проектах его не увидеть.
